# GithubPull

<b>This App shows PR of Github Open Projects in RecyclerView</b>


<b>-> Developed using :-</b>


<li>Kotlin Programming Language</li>

<li>MVVM Design Pattern </li>

<li>Data Binding</li>

<li>Pagination for Fetching data</li>
